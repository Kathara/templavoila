==============================================
Feature: Migrate template preview icons to FAL
==============================================

Description
===========

There is an update script (to be run in the install tool), to migrate all preview icons of template records from the good
old **/uploads/tx_templavoila/** to **/fileadmin/_migrated/templavoila/**.

Impact
======

Just one of a few steps to get rid of **/uploads/tx_templavoila/**. Run the migrate script and you are fine.
